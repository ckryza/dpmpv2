DPMP v2 Git workflow (safe)

1) Confirm clean:
   git status -sb

2) Backup files:
   cp -a dpmp/dpmpv2.py dpmp/dpmpv2.py.bak.$(date +%Y%m%d_%H%M%S)
   cp -a dpmp/config_v2.json dpmp/config_v2.json.bak.$(date +%Y%m%d_%H%M%S)

3) Edit only these:
   dpmp/dpmpv2.py
   dpmp/config_v2.json

4) Verify:
   .venv/bin/python -m py_compile dpmp/dpmpv2.py
   python3 -m tabnanny -v dpmp/dpmpv2.py

5) Commit + push:
   git add dpmp/dpmpv2.py dpmp/config_v2.json
   git commit -m "Describe change"
   git push

NOTE: dpmp_active_pool is label-based; use grep '^dpmp_active_pool' to see it.
