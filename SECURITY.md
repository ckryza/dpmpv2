# Security Policy

## Supported Versions
Only the latest released version of DPMP is supported with security updates.

## Reporting a Vulnerability
If you discover a security vulnerability, please do **not** open a public GitHub issue.

Instead, report it privately by:
- Using GitHubâ€™s Security Advisories feature, or
- Emailing the maintainer at: dpmp.project@gmail.com

Please include:
- A clear description of the issue
- Steps to reproduce, if possible
- logs (all events), if possible
- The affected version

## Disclosure Policy
Please allow reasonable time for the issue to be investigated and fixed before
any public disclosure.

Thank you for helping keep DPMP and its users secure.
