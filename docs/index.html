Dual Pool Mining Proxy (DPMP)

Deterministic Stratum v1 infrastructure for intelligent dual-pool mining.

Overview

DPMP is a high-reliability Stratum v1 proxy that sits between ASIC miners and two upstream pools. It time-slices work deterministically, preserves extranonce correctness, and evaluates fairness using difficulty-weighted acceptance rather than raw share counts.

Metrics & Stats

Worker and pool-level metrics with SumDiff tracking.

DPMP measures fairness using total accepted difficulty (SumDiff), normalizing across VarDiff and startup ramp differences between pools.

About

DPMP maintains concurrent upstream connections while serializing downstream work to prevent extranonce collisions and stale shares. Over time, accepted difficulty converges to the configured scheduler ratio.
